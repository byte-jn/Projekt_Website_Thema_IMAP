IMAP

Internet Message Access Protocol

Hauptfunktionen von IMAP:

Zugriff von mehreren Geräten: Mit IMAP können Nutzer ihre E-Mails von verschiedenen Geräten aus abrufen und verwalten, da die E-Mails auf dem Server gespeichert bleiben23.

Grundprinzip:

IMAP ist ein Protokoll zum Abrufen und Verwalten von E-Mails direkt auf dem Server. Es ermöglicht Clients den gleichzeitigen Zugriff auf dieselbe Mailbox, ohne dass die Nachrichten lokal dauerhaft gespeichert werden müssen.

Synchronisation und Statusverwaltung:

Da E-Mails auf dem Server verbleiben, bleiben Statusinformationen wie "gelesen", "ungelesen" oder "markiert" stets synchron – unabhängig davon, von welchem Gerät aus auf das Postfach zugegriffen wird.
